FAQ

  • Why is my gist's language classified as "Other"?
    Lepton depends on GitHub API to detect the language. If this fails, you can put "// vim: syntax=<your_language>" at the top of the gist to specify the language.
  • Why I can't search my gist's content?
    Currently, Lepton only supports the whole word search for the snippet description.
  • How to specify the title and keywords for my gist?
    GitHub Gist doesn't support title for its description section. Lepton makes enhancement by adding the title and keywords support. By following the format [title] description #keywords: key1 key2 in the gist description, the title and #keywords: key1 key2 will be seperated from the description.
  • How to ask for a new feature?
    Please submit a new feature request in the GitHub Issue page. Or, if you like, submit a pull request.
